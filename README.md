# Gamer Paradise

> Sistema de gerenciamento de inscrições para torneios de jogos.

> Tournament registration management system for games.

<br>

## Sobre / About

> Este projeto consiste em um sistema voltado ao gerenciamento de inscrições de torneios, permitindo organizar participantes, eventos e informações associadas. O foco do projeto está na integração entre diferentes camadas do sistema e no controle estruturado dos dados envolvidos.

> This project is a system focused on managing tournament registrations, allowing the organization of participants, events, and related information. Its focus is on integrating different system layers and structured data control.
